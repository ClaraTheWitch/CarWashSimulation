
PROJECT TITLE: Car Wash Simulation
PURPOSE OF PROJECT: To simulate the Squeaky Clean Car Wash with different wash 
                    times to see what differences the change makes and advise 
                    the car wash accordingly.
VERSION or DATE: 10/23/24
AUTHORS: Clara Siefke with assistance from Dr. Tasnim Gharaibeh
USER INSTRUCTIONS: The user may run the simulation by simply compiling the 
                    program and running the CarWashApplication class. The user 
                    may also look at the collected data in the pdf document in 
                    the file.
